The idea

Create a portfolio is an idea for a website or an assignment due for Business Web Technologies. This is the first assignment web fundamentals for unit ISYS3004.

What is the first thing you need to work on?

Content

Content is the most important element of a website. Written content, makes up for more than 90% of the Web The aim of design is to enhance the presentation of the content! Styling content will go a long way. The portfolio website whould include home page, about page, project page, contact page, and reflection page.

Objective

The objective of this assignment is to demonstrate the fundamental web technologies used in the development of a portfolio website. The students will use the web technologies they have learned throughout the course to create a portfolio website showcasing their projects and skills.
